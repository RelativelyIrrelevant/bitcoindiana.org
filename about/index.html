About bitcoINdiana

bitcoindiana.org is a simple collection of resources for people building a Bitcoin circular economy in Indiana. It uses public data from BTC Map and aims to make it easy to find places to spend sats.

Data attribution

Merchant locations come from BTC Map (which itself is built on OpenStreetMap data and community contributions). If something is wrong, the best fix is upstream so everyone benefits.
